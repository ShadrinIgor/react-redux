ТЗ Нужно сделать форму ввода, состоящую из двух полей (имя, фамилия). Имя не может содержать подряд двух букв 'а'. Фамилия может содержать дефис, но не больше одного во всем инпуте (не подряд, а суммарно может быть только 1 дефис). При сабмите происходит optimistic update, и посылается http запрос на сервер ("http://sample.com/api/v3/contact", как так такого эндпоинта нет, назад вернется ошибка, естественно). Cтэк технологий: React + Redux. Остальное по желанию.
